POST   /api/v1/assetcategories              → Create category
GET    /api/v1/assetcategories              → List categories
GET    /api/v1/assetcategories/active/list  → Active categories only
GET    /api/v1/assetcategories/:id          → Get category
PUT    /api/v1/assetcategories/:id          → Update category
DELETE /api/v1/assetcategories/:id          → Delete category

POST   /api/v1/itemtypes                    → Create item type
GET    /api/v1/itemtypes                    → List item types
GET    /api/v1/itemtypes/category/:catId    → Item types by category
GET    /api/v1/itemtypes/:id                → Get item type
PUT    /api/v1/itemtypes/:id                → Update item type
DELETE /api/v1/itemtypes/:id                → Delete item type
